### 2. Download Gradle Wrapper JAR

On first Gradle run, the wrapper will automatically download `gradle-wrapper.jar`. Alternatively:

```bash
cd android-app
./gradlew tasks  # Unix/Mac
# or
.\gradlew.bat tasks  # Windows
```

Or manually download from:
https://raw.githubusercontent.com/gradle/gradle/v8.0.0/gradle/wrapper/gradle-wrapper.jar

Save to: `android-app/gradle/wrapper/gradle-wrapper.jar`

### 3. Clone sparetools (Optional)

```bash

```

Then review using:
```bash

.\build\review-sparetools.ps1  # Windows
```

See `build/SPARETOOLS_INTEGRATION.md` for integration guide.

### 4. Build APK

**Debug APK:**

**Release APK:**
```bash
cd android-app

.\gradlew.bat assembleRelease  # Windows
```

Or use the convenience scripts:

## üêõ Troubleshooting

**Build fails with "gradle-wrapper.jar not found":**
- Run `./gradlew tasks` to auto-download, or manually download the JAR

**Build fails with missing icons:**
- Add launcher icons using Android Studio Image Asset Studio
- Or temporarily remove icon references from `AndroidManifest.xml`

**Gradle sync fails in Android Studio:**
- Ensure Android SDK is installed
- Check that `local.properties` points to correct SDK location (auto-generated by Android Studio)

**Dependencies not resolving:**
- Check internet connection
- Verify repositories in `settings.gradle` and `build.gradle`
- Try invalidating caches: File ‚Üí Invalidate Caches / Restart
