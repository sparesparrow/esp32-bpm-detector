Detect project characteristics using the dev-intelligence-orchestrator skill.

Run: `.claude/skills/dev-intelligence-orchestrator/scripts/detect_project_type.sh .`

This will identify:
- Languages (C++, Python, Kotlin, Java)
- Frameworks (PlatformIO, CMake, Conan, Gradle)
- Test frameworks (pytest, gtest, JUnit)
- Project nature (embedded, Android, desktop)
- FlatBuffers usage

Use this information to select appropriate tools and configurations.
